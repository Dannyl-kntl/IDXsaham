# IDX Stock Screener Telegram Bot 🤖

Bot screening saham IDX dengan 10 kriteria ala Goldman Sachs untuk investor ritel pemula.

## Fitur
- ✅ Screening 10 saham teratas IDX
- ✅ PER vs rata-rata sektor
- ✅ Analisis ROE, DER, Dividend Yield
- ✅ Pertumbuhan revenue & laba 5 tahun
- ✅ Rating Competitive Moat (Lemah/Sedang/Kuat)
- ✅ Rating Risiko skala 1-10
- ✅ Zona Entry & Stop-Loss otomatis

## Cara Install
1. `pip install -r requirements.txt`
2. Buat file `.env` dan isi `TELEGRAM_BOT_TOKEN=...`
3. `python bot.py`

## Perintah Bot
- `/screen` - 10 saham teratas
- `/detail BBCA` - Analisis lengkap
- `/search bank` - Cari saham
